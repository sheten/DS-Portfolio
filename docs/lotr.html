Film Review


University Project

Project Reason


Film Review page was constructed as a coursework for university fundamentals of computing module. We were asked to build a website using only HTML and CSS, as it was only things that we learnt by then. In website we had to review about film scenario, places, actors, I had to write about "The Lord of the Rings".

Project Development


Film Review pages are created with HTML and CSS using basics added text, styled it with colors, table, sizes, images and hrefs. Pages had container with two or three columns where information is presented.
Now I can see that it is hard to read, as colors are pastel and it matches background. Background itself is very dark and webpage lacks style for buttons or overall desing. But back then I thought it was okay.
