SoftConsulting


Side Project

Project Reason


SoftConsulting page was constructed to list part of possible systems and to increase clarity for companies and customers regarding of what type of systems exists in a marketplace. Page was crafted from basis using HTML5, CSS3, JavaScript, Bootstrap.

Project Development


SoftConsulting main page is constructed with HTML and CSS to see the list of existing systemsand let users choose on which system they want to learn more by clicking on it and getting redirected to its description page. From there users can read more, learn more and contact system providers for further information.
