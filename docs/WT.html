WiseTeam


Side Project

Project Reason


WiseTeam is company which is developing CRM and Project management system. This company's product introduction site was updated, because it had old design, so together with collegue we rebuilt the idea of how product information is presented.

Project Development


Wiseteam updates consist of changes with HTML, CSS and JQuery in navigation section, social media section and buttons.

Image sliders section were created using Jquery to show system functions and company's clients list. Also, were added
"Try Out Demo" form field, which allows customers to input neccesarry information, which would be send to company's server and instantly receive testing version of the system. On images of service added links, which would link to the description.
